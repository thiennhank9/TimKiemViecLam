Description

As a Misfit developer, you'll be contributing to the development of next-generation wearable products to improve peoples' health and wellbeing, while participating in the most innovative engineering team in Vietnam from feasibility to implementation.

Key Responsibilities:

  • Develop back-end features using Restful API, Microservice, AWS stacks technologies
  • Design and implement features that support client side and improve the availability, scalability, latency, and efficiency using Scala/ Nodejs/ Ruby/ Python
  • Develop and manage Java web, web service (API section): SOAP, REST, Spring, AngularJS
  • Work with other teams to provide relevant reports

Requirements

  • Bachelor's Degree or 3rd/4th year students in Infomation Techonology majority
  • Experience developing in Wordpress and other web design platforms
  • HTML, CSS and JavaScript experience a plus
  • Knowledge in CI/CD, Docker, and Restful API, AWS services
  • At least 6-month Full-time/Part-time Internship
  • Huge plus for Misfit: strong English communications skills

Benefits

  • Work in professional working environment
  • Experience team and company activities
  • Competitive internship allowance
